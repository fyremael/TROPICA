# TROPICA Colab Notebook Pack

This pack gives new operators and researchers a browser-first path through
TROPICA without requiring local setup.

| Notebook | Audience | Purpose |
| --- | --- | --- |
| [01_operator_onboarding.ipynb](01_operator_onboarding.ipynb) | New operators | Install TROPICA, run the evidence suite, inspect the generated report, and understand the pass/fail contract. |
| [02_benchmark_suite.ipynb](02_benchmark_suite.ipynb) | Maintainers and evaluators | Run the benchmark/report suite, summarize gate results, display all dashboards including unified traces, and package artifacts for sharing. |
| [03_researcher_showcase.ipynb](03_researcher_showcase.ipynb) | Researchers | Compile bounded tool-call specs into real tokenizer masks, drive the decoder with offline logit providers, inspect traces, and exercise negative controls. |

Open directly in Colab:

- [Operator onboarding](https://colab.research.google.com/github/fyremael/TROPICA/blob/main/notebooks/01_operator_onboarding.ipynb)
- [Benchmark suite](https://colab.research.google.com/github/fyremael/TROPICA/blob/main/notebooks/02_benchmark_suite.ipynb)
- [Researcher showcase](https://colab.research.google.com/github/fyremael/TROPICA/blob/main/notebooks/03_researcher_showcase.ipynb)

The notebooks install from the GitHub repository when opened in a plain Colab
runtime, or install editable from the current checkout when run inside a cloned
source tree.
